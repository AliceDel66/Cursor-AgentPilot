# Cursor 新手手册

> 本章为 Sprint 1 骨架，正文将在 Sprint 2 补全。目标：新手 30 分钟内理解 Cursor 基本工作流，并完成第一次 AI 辅助修改。

## 1. Cursor 是什么

一句话说清 Cursor 的定位：从代码编辑器演进为 agentic coding 工作台，以及它与传统 IDE + Copilot 的差异。

## 2. 安装与初始配置

覆盖下载安装、登录、模型选择、导入 VS Code 配置等首次启动必做项。

### 2.1 安装与登录

安装步骤与账号/订阅说明。

### 2.2 必要设置

新手推荐的初始设置：模型、隐私、快捷键。

## 3. 界面速览

介绍编辑器、Chat 面板、Agent 窗口、终端等核心区域，让新手知道"东西在哪"。

## 4. 核心能力地图

概览 Tab 补全、Inline Edit、Chat、Agent 四种基础交互，以及各自适合的场景。

## 5. 第一次 AI 辅助修改（实操）

手把手完成一次小功能修改：提需求、看 diff、接受/拒绝、验证结果。

## 6. 怎么写好 prompt

常见开发任务的 prompt 写法：给上下文、给范围、给验收标准，附正反例。

## 7. 新手常见错误

列举新手高频踩坑（把 Agent 当自动驾驶、不看 diff 直接接受、任务范围过大等）与纠正方式。
